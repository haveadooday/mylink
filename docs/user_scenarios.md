# 마이링크 (MyLink) 사용자 시나리오 (User Scenarios)

## 1. 방문자 (Visitor): 마이링크 페이지 방문 과정
- 사용자는 **특정 크리에이터의 다양한 운영 채널을 한눈에 확인**하기 위해 **공유받은 마이링크 고유 URL(예: 마이링크.com/닉네임)로 접속**한다.
- 사용자는 **자신이 관심 있는 콘텐츠(포트폴리오, 영상 등)로 이동**하기 위해 **자동 연동된 파비콘이 함께 표시된 링크 버튼을 클릭**한다.
- 사용자는 **해당 크리에이터의 인스타그램이나 이메일 등으로 직접 접근**하기 위해 **프로필 상/하단에 배치된 소셜 퀵 링크 아이콘을 누른다**.
- 사용자는 **유용하다고 생각되는 이 프로필을 타인과 공유**하기 위해 **화면에 제공되는 '링크 간편 복사' 버튼을 터치하여 주소를 복사**한다.

## 2. 소유자 (Owner): 링크 추가/수정/삭제 (관리) 과정

### 2.1. 초기 접근
- 사용자는 **자신의 링크 정보를 관리하는 마이페이지에 진입**하기 위해 **Firebase 기반의 구글 소셜 로그인 버튼을 클릭하여 로그인**한다.

### 2.2. 링크 추가 과정
- 사용자는 **프로필에 새로운 외부 방문 경로를 추가**하기 위해 **'링크 추가' 버튼을 클릭**한다.
- 사용자는 **방문자가 직관적으로 해당 링크의 목적을 이해하게** 하기 위해 **입력칸에 '타이틀'과 실제 이동할 'URL'을 기입**한다. (이때 시스템이 URL을 감지하여 파비콘을 자동으로 불러온다.)

### 2.3. 링크 수정 및 정렬 과정
- 사용자는 **입력했던 링크의 글귀나 목적지 주소를 변경**하기 위해 **해당 링크 블록의 활성화된 텍스트 필드를 클릭하여 내용을 재작성**한다.
- 사용자는 **현재 시점에는 필요 없는 링크를 삭제하지 않고 깔끔하게 감추**기 위해 **해당 링크 우측에 위치한 활성화 On/Off 토글을 꺼둔다(Off)**.
- 사용자는 **가장 집중적으로 홍보해야 할 최우선 링크를 상단에 배치**하기 위해 **해당 링크 블록을 마우스로 잡고 위아래로 끌어다 놓는다(드래그 앤 드롭)**.

### 2.4. 링크 삭제 과정
- 사용자는 **더 이상 유효하지 않거나 영구적으로 불필요해진 링크를 완전히 지우**기 위해 **해당 링크 블록에 있는 '삭제(휴지통)' 아이콘을 클릭하여 제거**한다.
